Prerequisites

Before using Semant, ensure you have:

  • Python 3.11+
  • Qdrant vector database (for embeddings)
  • OpenAI API key (for AI features)
  • Google Cloud credentials (optional, for GCS)

Quick Start

# 1. Start Qdrant docker run -p 6333:6333 qdrant/qdrant # 2. Set environment variables export OPENAI_API_KEY="your-key-here" export GCS_BUCKET_NAME="your-bucket" # 3. Start the server python main.py # 4. Check health curl http://localhost:8000/api/health

âœ¨ Best Practices

Agent Development

  • Inherit from BaseAgent: Always extend BaseAgent for proper integration
  • Define capabilities: Clearly declare what your agent can do
  • Use async methods: All agent methods should be async
  • Handle errors gracefully: Implement proper error handling and logging
  • Document your agent: Add comprehensive docstrings and examples

Workflow Management

  • Use the registry: Register agents for proper discovery
  • Implement timeouts: Set reasonable timeouts for operations
  • Monitor performance: Use the monitoring dashboard to track metrics
  • Handle failures: Implement retry logic and fallback mechanisms

Knowledge Graph Usage

  • Use SPARQL carefully: Optimize queries for performance
  • Validate data: Check data before adding to the graph
  • Handle concurrency: Use proper locking for concurrent updates
